# gzsize(1) -- Get the uncompressed size of an archive

## SYNOPSIS

`gzsize` <file> [<file> ...]

## DESCRIPTION

Measure the uncompressed size of a GZipped file.

## OPTIONS

  * `-h`:
    Print the help and exit

  * `-H`:
    Output an human-readable size

  * `-v`:
    Print the version and exit


## EXAMPLES

  * Check a file:
    `gzsize myfile.gz`
